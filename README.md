# 🖥️ Media Interaktif SMP/MTs

**Platform belajar interaktif untuk SMP/MTs — Kurikulum Merdeka Fase D**

## 📚 Fitur

- **15 Mata Pelajaran** (PAI & Umum) untuk Kelas 7, 8, 9
- **Latihan Interaktif**: Kuis PG, Isian Singkat, Benar/Salah, Flashcards, Drag & Drop
- **Simulasi**: Konversi Biner, Gerbang Logika, Visualisasi Sorting, Jaringan, Enkripsi Caesar Cipher
- **Tools**: Flowchart Builder, Teka-Teki Silang, Puzzle Kode, Glosarium Multi-Mapel
- **Progress Tracker** dengan badge pencapaian
- **Dark Mode** otomatis
- **Fully offline** — tidak perlu server atau database

## 🚀 Teknologi

Murni **HTML + CSS + JavaScript** — tidak ada dependency, build tools, atau backend.
